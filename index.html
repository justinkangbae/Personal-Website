Portfolio

Here are some of the applications I have worked on and would like to share with you. These applications consist of projects that have utility and some that I created just for fun.

YelpCamp

Web app designed to showcase different campgrounds that users can share and post about. Implemented with RESTful routing and authentication with login and register functionality. Coded with Node.js, Express, MongoDB, EJS, and others.

RGB Color Game

Simple game coded in HTML, CSS, and JavaScript in which the player must match a randomly generated RGB color to one of either three or six squares based on chosen difficulty (Easy or Hard).

To-Do List

To-do list coded in HTML, CSS, and jQuery where the user can submit new list items by typing it in and confirming with “Enter” keypress, cross off completed to-do items, and delete list items.
